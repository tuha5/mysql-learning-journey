# PersonalLibraryTrackerMySql
# Day 1: Personal Library Tracker

## Project Overview
A MySQL database project to track personal book collection with reading status.

## Database Schema
- **Database:** library_db
- **Table:** books
- **Columns:** book_id, title, author, genre, year_published, read_status, date_added

## Features
- Track reading progress
- Categorize books by genre
- Filter by read/unread status
- Sort by publication year

## How to Use
1. Run database_schema.sql to create database and table
2. Run sample_data.sql to insert sample data
3. Use queries.sql for example operations

## What I Learned
- CREATE DATABASE/TABLE
- INSERT data
- SELECT queries with WHERE, ORDER BY, GROUP BY
- UPDATE and DELETE operations
